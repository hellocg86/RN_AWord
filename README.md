# RN_AWord
react native 编写的demo
代码写的很渣，请见谅，主要看看性能，看看和原生的差距。
还有数据是采集了别人的。。。。。。。

<p1>
首先在项目目录
1: npm install</br>
2: cd server && cd bin</br>
3: node www</br>
4: 在主目录下 cd ios</br>
5: pod install</br>
6: 用xcode 打开 testdemo.xcworkspace</br>
</p1>
