# RN_AWord
react native 编写的demo
代码写的很渣，请见谅，主要看看性能，看看和原生的差距。
还有数据是采集了别人的。。。。。。。


首先在项目目录
1: npm install
2: cd server && cd bin
3: node www
4: 在主目录下 cd ios
5: pod install
6: 用xcode 打开 testdemo.xcworkspace
